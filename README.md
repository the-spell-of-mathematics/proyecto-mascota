# 🐻 Mi Osito Virtual: Kodiak 🍯

¡Bienvenido a mi proyecto de mascota virtual! 🎨✨ Aquí aplico conceptos fundamentales de **Programación Orientada a Objetos en JavaScript** para darle vida a Kodiak, un tierno osito estilo acuarela al que hay que cuidar, alimentar y mimar.

---

### 🎮 Etapas Superadas del Bootcamp

A lo largo de este proyecto, construí la lógica paso a paso hasta llegar a la interfaz gráfica completa:

| Nivel | Archivo | Misión |
| :---: | :--- | :--- |
| 🟢 1 | `mascota.js` | **Nace tu mascota:** Creación del objeto base y sus propiedades (stats del 0 al 100). |
| 🟢 2 | `mascota.js` | **El método `describir()`:** Uso de la palabra clave `this` para leer propiedades internas. |
| 🟢 3 | `mascota.js` | **`alimentar()`:** Modificar el estado de la mascota usando `Math.max` para establecer límites. |
| 🟢 4 | `mascota.js` | **`jugar()` y `dormir()`:** Cambiar múltiples stats a la vez con topes usando `Math.min`. |
| 🟢 5 | `mascota.js` | **Ver stats:** Recorrer y filtrar las propiedades numéricas del objeto con `Object.entries`. |
| 🟢 6 | `mascota.js` | **Clon VIP:** Copiar el objeto sin alterar el original usando *spread operator* (`...`) y destructuring. |
| 👾 JEFE | `index.html` / `CSS` | **Interfaz Gráfica:** Conectar la lógica de JS con el DOM, manipulando eventos, temporizadores y GIFs. |

---

### 🛠️ Herramientas y Tecnologías Usadas

Para lograr la estética de tonos pastel y la funcionalidad interactiva, utilicé:

* **HTML5:** Estructura semántica del proyecto.
* **CSS3:** Estilos en paleta pastel, tipografía cómic redondeada (*Fredoka*) y la propiedad `mix-blend-mode: multiply` para eliminar el fondo blanco de los GIFs.
* **JavaScript (Vanilla):** Programación Orientada a Objetos, manipulación del DOM, `setTimeout` y `setInterval`.
* **Git & GitHub:** Control de versiones y despliegue del repositorio.
